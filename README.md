# Wholesale_Dataset-Visualization-in-Tableau
Tableau Project_Wholesale_Purchase Dataset

Project Overview

This project uses Tableau to analyze a wholesale purchase dataset and present customer spending behavior through interactive dashboards and visual storytelling.
The objective was to explore how customers spend across product categories and regions, and turn raw purchase data into insights that support sales, marketing, and inventory decisions.

🎯 Key Questions Explored
1.Which product categories generate the highest spending?
2.How do purchasing patterns vary by customer type and region?
3.What is the distribution of customer purchases?
4.How can visual insights guide business strategy?


📈 Dashboards & Sheets Included

This project contains the following Tableau visualizations:

-Customer Type By Region — compares customer segments across regions
-Purchase by Product Category — highlights top revenue-driving categories
-Purchase Distribution — shows how spending is spread across customers
-Donut Chart Visualization — summarizes category contribution
-Wholesale Dashboard — combines all insights into one interactive view
